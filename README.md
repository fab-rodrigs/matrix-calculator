# matrix-calculator
This calculator does basic operations for matrices without the aid of algebra libraries. Currently it can do matrix determinant, transpose and inverse.

## Working example

```
--> Operacoes com matrizes <--

Numero de linhas: 2
Numero de colunas: 2

C11: 1
C12: 2
C21: 3
C22: 4

| 1 2 |
| 3 4 |

Calculo:
1.Determinante
2.Transposta
3.Inversa
0.Sair
```
